# 適應期注意事項 — 專案說明

## 專案概述

Dr.Li 科學體重管理課程的學員專屬教材，為單頁靜態 HTML 網頁，部署於 GitHub Pages。

**檔案**：`適應期注意事項.html`（唯一主檔）

## 頁面內容

學員進入計畫第一個月的三大核心習慣：

| 章節 | 錨點 | 重點 |
|------|------|------|
| ① 喝足夠的水 | `#water` | 公式：體重 × 30 + 500 mL（最低）/ × 50 mL（理想） |
| ② 進食順序 | `#order` | 口訣：水 → 菜 → 肉 → 飯 → 果 |
| ③ 主食份量 | `#staple` | 每餐上限 2 份，不可累加，每日至少一餐需有主食 |

## 技術架構

- **純 HTML + CSS**，無外部 JS 框架
- 字型：Google Fonts（Noto Sans TC + Inter）
- 版面：CSS Grid / Flexbox，RWD 斷點 500–600px
- 色彩系統：CSS 自訂屬性定義於 `:root`（`--primary`、`--accent`、`--orange` 等）
- 目錄列固定置頂（`position: sticky; top: 0; z-index: 100`）

## 主食份量對照（核心資料）

| 食物 | 1 份份量 |
|------|---------|
| 白米飯 / 糙米飯 | ¼ 碗（熟重約 50g） |
| 白粥 / 稀飯 | ½ 碗（熟重約 125g） |
| 白麵條 / 烏龍麵 / 冬粉 | ½ 碗（熟重約 60g） |
| 薄片吐司 / 全麥吐司 | 1 片（約 25–30g） |
| 饅頭 | ⅓ 個（約 30g） |
| 燕麥片（乾） | 3 湯匙（約 20g） |
| 地瓜 | ½ 個小（約 55g） |
| 馬鈴薯 | ½ 個中（約 90g） |
| 玉米 | ⅔ 根（約 85g） |
| 南瓜 | ¼ 個小（約 85g 熟） |

1 份主食 ≈ 含 15g 醣質 ≈ 70 大卡

## 修改注意事項

- **數值異動**（公式、份量克重）需同步更新頁面內所有出現處
- 根莖類食物（地瓜、玉米、南瓜、馬鈴薯、山藥、芋頭、栗子）**計入主食份數**，非蔬菜，修改時注意分類正確性
- 水果建議時間（10:00–11:00、14:30–16:00）若調整需一併修改
- 頁尾聲明「僅供學員使用，請勿對外分享」，屬版權聲明，不可刪除
